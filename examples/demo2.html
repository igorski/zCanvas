zCanvas demo #2: Shoot 'em up with fast collision detection

Demo features : multiple layers of parallax scrolling, endless waves of enemies, spritesheet explosions and tons of bullets using the high performing collision detection routine (which is less precise but adequate in this context). Furthermore, framerate is adjustable at runtime while game speed is maintained. Use arrow keys and spacebar to shoot. A complete game based on this demo, written using zCanvas can be found here.
